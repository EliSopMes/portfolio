Join Immersive

Chrome extension to make reading German text easier for language learners.

Learning German through content you actually care about.

This is a Chrome extension coded in JavaScript hosted in the Chrome extension store. It also connects to Netlify hosted functions that take care of handling the API calls. The purpose is to be able to simplify German text based on your level (A1 - C1), so you can interact with content that you actually care about in the language that you are learning. You are able to simplify, translate, listen to the pronounciation of individual German words or entire paragraphs, as well as save the vocabulary that you want for practicing with it later.

tech-ERS

Startup project sponsered by the Berlin Startup Stipendium through an incubator program. Focus on helping secondary school teachers save time in their class preparation and work overall.

Empowering teachers with AI

I built this project using a decoupled Vue.js frontend and a Ruby on Rails API backend. It features a Learning Hub and an Editing Tool for teachers.

1. Learning Hub – A curated selection of YouTube content where educators learn about AI topics like prompting, bias, and LLMs.

The content is collected on Notion and accessed through their API, making it independent from development.

2. Editing Tool – Designed to help teachers repurpose PDFs. Users can drag and drop files, which are processed through Adobe's API to extract metadata. Images are clipped, and text elements are analyzed via OpenAI's API to generate summaries, keywords, and titles for easier organization.

The Vue frontend enables element-by-element editing, including text modification, repositioning, and deletion.

Additional features include Document Storage, user authentication (email, Google, Microsoft) and profile customization. To foster community, we host workshops/webinars (integrated with Eventbrite) and provide a WhatsApp group for teachers to connect.

Game on!

Final group project in my Web Development bootcamp. The idea was to build a platform where people can easily connect to do leisure activities together.

Meet up for sports activities

Ruby on Rails project hosted on Heroku build together with 3 other students. The platform includes functionality of creating, showing, updating, deleting casual sporting events (e.g. football match in a park) for others to join. We also included a map using Mapbox API with pins for the event locations, a chat using ActionCable to connect with other players joining the same event and a calendar overview to keep track of upcoming events.

This project was not only about the functionality but also about coordinating within a team by using Github.
